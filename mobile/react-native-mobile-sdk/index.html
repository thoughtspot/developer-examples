Welcome to ThoughtSpot + React Native Embed

This is a simple example of how to embed ThoughtSpot in a React Native app.

Step 1: Install the ThoughtSpot React Native SDK.

Step 2: Obtain your ThoughtSpot host and credentials.

Step 3: Create a Liveboard in ThoughtSpot.

Step 4: Authenticate using Trusted Authentication or other auth method using init() method.

Step 5: Use the Liveboard ID in the React Native embed.

Step 6: Customize the look and feel using custom CSS.

Click "Start" to see the embedded Liveboard on the expo sandbox.
